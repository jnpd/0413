# 通用契约索引

## 1. 说明

当前仓库不再单独保留顶层通用契约目录。

通用契约已按主题拆分到现有 `00-07` 目录下，原则如下：

- 项目级文档优先于通用契约
- 只保留现有顶层目录结构，不新增新的数字目录
- 需要引用通用契约时，按下表进入对应分区

## 2. 优先级

```text
change/contracts 或后续补齐的规格真源
> 项目级 docs
> 分散落盘的通用契约
> 外部 starter 模板
```

## 4. 使用方式

- 写项目规则时，先读项目级正文，再补读对应通用契约。
- 需要做接口、提交、错误、分页等统一约束时，优先从本索引跳转，不要再新建一套通用规范。
- 如果后续新增通用契约，也继续按主题拆进现有 `00-07`，不要恢复顶层独立目录。
