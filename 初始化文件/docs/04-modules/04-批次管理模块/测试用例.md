# 04 批次管理模块测试用例

## 1. 适用范围

覆盖备货单号分组、测试批次创建与编辑、重命名、删除守卫及下游选择口径。

## 2. 用例清单

| 编号 | 类型 | 场景 | 预期结果 |
|---|---|---|---|
| `BMG-01` | smoke | 加载列表并按测试批号或备货单号搜索 | 分组行与批次明细行可展开；搜索结果回到第一页 |
| `BMG-02` | 主链路 | 新增合法备货单号 | 备货单号符合 `2603-001` 格式；保存成功后列表可见 |
| `BMG-03` | 边界 | 新增不符合 `^\\d{4}-\\d{3}$` 的备货单号 | 保存被阻断，前后端格式校验一致 |
| `BMG-04` | 主链路 | 在某个备货单号下新增测试批次 | 调用 `batch/create` 后自动生成下一个测试批号，并跳转到批量测试页 |
| `BMG-05` | 主链路 | 编辑备货单号并重命名 | 该组下所有测试批次前缀同步改写，`P001/P002...` 后缀保持不变 |
| `BMG-06` | 边界 | 已存在历史快照时重命名备货单号 | 先触发二次确认，再决定是否继续重命名 |
| `BMG-07` | 边界 | 删除仍存在有效测试批次的备货单号 | 删除被阻断，不能误删分组头 |
| `BMG-08` | 边界 | 删除已生成历史快照、已绑定检验台数据或已产生包装记录的测试批次 | 删除被阻断，包装记录优先按关系表判断 |
| `BMG-09` | 回归 | 下游模块读取 `prep/select-list` 与 `batch/select-list` | 仓库、批量测试、包装模块下拉数据口径一致 |
| `BMG-10` | 回归 | 新增、重命名、删除等写操作提交中 | 页面与弹窗显示整体阻断式 `loading`，不能重复提交 |

## 3. 高风险回归触发

- 备货单号或批次号编码规则变更
- 批次创建后跳转行为变更
- 批次删除守卫依赖关系变更
- `select-list` 下游接入逻辑变更
