# 06 测试历史模块测试用例

## 1. 适用范围

覆盖历史分页、历史详情、检验台 Excel 上传、手动绑定、删除守卫与快照删除边界。

## 2. 用例清单

| 编号 | 类型 | 场景 | 预期结果 |
|---|---|---|---|
| `HIST-01` | smoke | 进入历史列表并按表号、备货单号、测试批号、表具名称搜索 | 列表分页正常；1-14 位纯数字表号查询前自动补零到 14 位 |
| `HIST-02` | 主链路 | 打开某条历史详情 | 详情页读取快照数据，不回退到实时批量测试当前态 |
| `HIST-03` | 边界 | 上传非 `.xlsx` 文件或缺列模板文件 | 上传被拒绝，并提示模板要求 |
| `HIST-04` | 边界 | 上传 Excel 时某个表号不是正好 3 个流量点或缺少 `2500/40/25` | 上传被拒绝，并给出可读错误原因 |
| `HIST-05` | 回归 | 确认上传期间尝试重复提交或关闭弹窗 | 按钮进入 `loading`；显示“数据正在上传中”提示；禁止重复提交和关闭弹窗 |
| `HIST-06` | 主链路 | 选择 1 个平台表号和 1 条检验台记录，执行“匹配 3 条” | 前端高亮对应 3 条流量点；调用 `/bench/bind` 成功建立绑定 |
| `HIST-07` | 边界 | 删除已被历史快照引用的检验台记录 | 删除被阻断，不能误删记录头与原始行 |
| `HIST-08` | 主链路 | 删除单条历史快照 | 仅软删当前 `snapshotId` 及其历史明细和绑定关系，不删除全局检验台原始记录 |
| `HIST-09` | 回归 | 历史列表或详情中的纯数字表号展示 | 平台历史明细与检验台行统一按 14 位展示 |

## 3. 高风险回归触发

- Excel 模板列或三流量点规则变更
- `bench/bind`、`bench/remove`、`remove` 接口变更
- 历史快照删除范围变更
- 表号补零口径变更
