## 1. 后端前置

## 2. 本地必确认项

- 服务端口：`8010`
- MySQL 配置已可用
- 仓库相关 SQL 已执行
- 测试账号可登录

## 3. 登录口径

- 登录接口：`POST /blade-auth/oauth/token`
- 默认租户：`000000`
- 密码加密：`04 + sm2.doEncrypt(password, publicKey, 0)`

## 4. 业务请求头口径

- `Authorization: Basic c2FiZXIzOnNhYmVyM19zZWNyZXQ=`
- `Blade-Auth: bearer <token>`
- `Blade-Requested-With: BladeHttpRequest`

## 5. 最小 smoke 顺序

1. 登录拿 token
2. 调 `GET /blade-wms/warehouse/stat`
3. 调 `GET /blade-wms/warehouse/page`
4. 再根据需要测 `submit/remove/outbound`

## 6. 常见失败优先排查项

- 假登录，没有真实 token
- 少了 `Blade-Auth`
- 密码加密模式错误
- MySQL 脚本未执行
